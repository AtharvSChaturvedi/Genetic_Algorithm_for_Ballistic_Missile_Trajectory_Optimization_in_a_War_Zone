# Genetic_Algorithm_for_Ballistic_Missile_Trajectory_Optimization_in_a_War_Zone
GA-based 3D missile trajectory optimizer for war-zone navigation. Minimizes fuel cost &amp; instability while avoiding 4 enemy defense zones under altitude &amp; G-force constraints. Uses tournament selection, crossover, Gaussian mutation &amp; elitism, 200 candidates evolved over 500 generations.
